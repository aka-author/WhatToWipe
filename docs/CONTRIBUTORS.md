# Contributors — Erase & Rewrite

This file satisfies LEGALSPEC LS-110–LS-113. It lists people who have contributed merged work to the Program or to normative documentation (excluding owner-only FS edits).

Each entry gives a **display name** (real name or anonymous pseudonym) and a **short description** of inputs. See [specs/legalspec.md](specs/legalspec.md) §4.1 for contributor obligations.


## How to add an entry

When a non-trivial contribution is merged:

1. Add a row to the table below (or a new subsection if the table grows large).
2. Use the contributor's chosen real name **or** an acceptable anonymous pseudonym.
3. Describe the contribution briefly (area, feature, or documents — not every commit).
4. Ensure GPLv3-or-later agreement is recorded per LS-108 (for example DCO `Signed-off-by` in the commit message).

The principal copyright holder may reject offensive, discriminatory, or otherwise unacceptable pseudonyms (LS-111).
